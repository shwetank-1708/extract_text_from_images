## Project Title: - Extract Text From Images (OCR App)

## Description: -

This project is to generate a platform, where a user will drag or upload an image file of "jpg, jpeg and png" format. And this app will convert the text presented in the image to into a text format, which a user can copy and then use it as per their need.

## Tech Stack: -

Python
Streamlit
PaddleOCR
OpenCV
Pillow
NumPy

## Features: -

Extract text from images using PaddleOCR
Image preprocessing (grayscale, contrast, sharpening)
OCR accuracy evaluation using CER and WER metrics
Confidence score display
Download extracted text as .txt or .json
Interactive Streamlit web interface
